# PhotoAlbumLT
A simple console application that displays photo ids and titles in an album through online web service, implemented basic unit testing.

Language: C#

IDE: Visual Studio 2017

## How to Use?
(1) Git clone repo

(2) Open project solution PhotoAlbumLT.sln

(3) Build & Run
